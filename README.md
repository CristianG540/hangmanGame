# Getting Started

### INSTALLING GLOBAL DEPENDENCIES

```sh
$ npm install -g sails
$ npm install -g bower
```

### INSTALLING PROJECT DEPENDENCIES
In the root of the project run:
```sh
$ npm install
```
Now, under the assets directory, run:
```sh
$ bower install
```

### Lift Off!
In the root of the project run:
```sh
$ sails lift
```
